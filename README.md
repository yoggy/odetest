odetest
====
ODE(Open Dynamics Engine)
  - http://ode.org/

![img01.gif](img01.gif)

    
How to
----

    $ sudo apt-get install libode-dev cmake
	$ git clone https://github.com/yoggy/odetest.git
    $ cd odetes
    $ cmake .
    $ make
    $ ./odetest02
